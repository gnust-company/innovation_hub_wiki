# Changelog

## Recent Changes

### v1.0 — Initial Release
- Problem Feed: CRUD, categories, privacy, comments, reactions.

- Events: Teams, ideas, scoring, FAQ, awards.
- Authentication: JWT, roles, profile management.
- Dashboard: User dashboard, admin analytics.
- Notifications: 15+ notification types, click-to-navigate.

- i18n: Vietnamese + English.
- Rich text: TipTap editor with image support.
