# MOC Dashboard

Dashboard hiển thị thống kê, analytics và hoạt động gần đây.

## Các khía cạnh
- [[05_Dashboard/Statistics]] — Các chỉ số thống kê (problems, ideas, users, engagement)
- [[05_Dashboard/OKR_Tracking]] — Theo dõi OKR (50 ideas/year, 70% engagement)

## Hai loại Dashboard

### User Dashboard (`/dashboard`)
- Dành cho mọi user.
- Stat cards cá nhân (problems, rooms, events).
- Trending ideas.
- Recent activity feed.
- Top contributors.
- Quick actions.

### Admin Analytics (`/admin`)
- Dành cho admin.
- Toàn bộ thống kê (bypass privacy).
- Charts: activity over time, category distribution.
- User management (`/admin/users`).
- Xem [[09_Admin_Guide/MOC_Admin]].

## Mối liên hệ
- Stats từ [[01_Problem_Feed/MOC_Problem_Feed]] reactions, comments.
- Stats từ [[02_Idea_Lab/MOC_Idea_Lab]] votes, ideas.
- Stats từ [[03_Events/MOC_Events]] participation, scoring.
