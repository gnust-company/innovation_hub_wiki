# OKR Tracking

## Mục tiêu OKR
Innovation Hub hỗ trợ KR3 của tổ chức:

| KR | Mục tiêu | Cách đo |
|----|----------|---------|
| KR1 | 50 ideas/năm | Đếm tổng ideas created (Room + Event) |
| KR2 | 70% engagement rate | (Số user active) / (Tổng user) |
| KR3 | 20% converted to pilot | Ideas có status "submitted" hoặc higher |

## Tracking qua Dashboard
- Admin Analytics hiển thị các chỉ số tổng hợp.
- Xem [[05_Dashboard/Statistics]] cho chi tiết từng metric.
- Date range filtering để so sánh theo kỳ.
