# Quick Start

## Đăng nhập lần đầu

### Nếu chưa có tài khoản
1. Vào trang đăng ký (`/register`).
2. Nhập username, password.
3. Optional: email, full name, team.
4. Click "Đăng ký" → tự động đăng nhập.

### Nếu đã có tài khoản
1. Vào trang đăng nhập (`/login`).
2. Nhập username + password.
3. Click "Đăng nhập".

## Cấu hình Profile
1. Click avatar/username ở sidebar → Settings (`/settings`).
2. Cập nhật:
   - **Full name**: Tên hiển thị.
   - **Email**: Email liên hệ.
   - **Team**: Bộ phận/team bạn thuộc về.
   - **Avatar**: Upload ảnh đại diện (max 10MB, JPEG/PNG/WebP).
3. Click "Lưu".

## Navigation cơ bản
- **Sidebar** bên trái chứa menu chính.
- **Notifications** (chuông) ở header — click để xem thông báo.
- Mỗi page có breadcrumb để navigate.
