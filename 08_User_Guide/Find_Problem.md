# Find Problem

## Vào Problem Feed
Click **"Problems"** ở sidebar → `/problems`.

## Tìm kiếm & Filter
- **Search bar**: Tìm theo tiêu đề, nội dung.
- **Category**: Process, Technical, People, Tools, Patent.
- **Status**: Open, Discussing, Brainstorming, Solved, Closed.
- **Author**: Lọc theo người đăng.
- **Date range**: Từ ngày - Đến ngày.
- **My Problems**: Toggle chỉ xem bài của mình.
- **Sort**: Mới nhất, cũ nhất, nhiều like nhất, nhiều comment nhất.

## Xem chi tiết Problem
1. Click vào problem card.
2. Xem nội dung rich text.
3. Xem reactions (like, dislike, insight).
4. Xem comments và replies.
5. Nếu problem link với Room → hiển thị link.

## Tương tác
- **Reaction**: Click icon like/dislike/insight. Xem [[01_Problem_Feed/Reactions]].
- **Comment**: Thêm bình luận hoặc reply. Xem [[01_Problem_Feed/Comment_Threading]].
- **Tạo Room**: Click "Tạo Room" để brainstorm. Xem [[02_Idea_Lab/Create_Brainstorm_Room]].
