# Glossary

## Thuật ngữ nền tảng

| Thuật ngữ | Định nghĩa |
|-----------|-----------|
| **Innovation Hub** | Nền tảng nội bộ quản lý đổi mới sáng tạo |
| **Problem Feed** | Newsfeed đăng và thảo luận vấn đề |
| **Idea Lab** | Không gian brainstorming với rooms và ideas |
| **Room** | Phòng brainstorming chứa ideas, có thể link với Problem |
| **Idea** | Ý tưởng đóng góp trong Room |
| **Event** | Sự kiện đổi mới sáng tạo (cuộc thi) |
| **Team** | Đội tham gia Event |
| **Kanban Board** | Bảng drag-drop hiển thị ideas theo status |
| **OKR** | Objectives and Key Results — Khung theo dõi mục tiêu |
| **KR** | Key Result — Kết quả chính |

## Roles & Permissions

| Thuật ngữ | Định nghĩa |
|-----------|-----------|
| **Member** | Thành viên thường, role mặc định |
| **Admin** | Quản trị viên, quyền đầy đủ |
| **Team Lead** | Leader trong Event Team (event-specific) |

## Technical

| Thuật ngữ | Định nghĩa |
|-----------|-----------|
| **JWT** | JSON Web Token — Chuẩn xác thực |
| **Access Token** | Token ngắn hạn (30 phút) cho API requests |
| **Refresh Token** | Token dài hạn (7 ngày) để lấy access token mới |
| **TipTap** | Rich text editor framework |
| **JSONB** | Kiểu dữ liệu JSON trong PostgreSQL |
| **MinIO** | Object storage cho avatars và files |
| **Zettelkasten** | Phương pháp ghi chú atomic, linked |
| **MOC** | Map of Content — Mục lục trong Zettelkasten |
| **Wikilink** | Cú pháp liên kết `[[Tên_Note]]` trong Obsidian |

## Status Workflows

| Thuật ngữ | Định nghĩa |
|-----------|-----------|
| **Open** | Problem mới đăng, chờ thảo luận |
| **Discussing** | Problem đang có bình luận |
| **Brainstorming** | Problem đã có Room brainstorm |
| **Draft** | Idea mới tạo |
| **Refining** | Idea đang tinh chỉnh |
| **Reviewing** | Idea đang xem xét |
| **Submitted** | Idea đã nộp / Event active |
| **Solved** | Problem đã giải quyết |
| **Closed** | Problem/Event đã đóng |
